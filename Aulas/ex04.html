Formatando Strings

var s = 'JavaScript'

  • 'Eu estou aprendendo s'
    • Nesse caso a escrita JavaScript não aparecerá.
  • 'Eu estou aprendendo' + s
    • Aqui está sendo usado a CONCATENAÇÃO e aparecerá a escrita JavaScript.
  • `Eu estou aprendendo ${s}`
    • Nesse caso está sendo usado Placeholder($) junto com crase. Essa técnica chama-se template string e é usada para fazer interpolação.
